    • 看Python基本教程

    1.Python内置的有序集合list和tuple之间的区别，list变量与tuple变量均为指针，但是前者可变，后者不可变。嵌入list变量的tuple变量虽然可以随着list变量的变化而变化，但是tuple变量作为指针指向的那个地方是不变的。

    >>> t = ('a', 'b', ['A', 'B'])
>>> t[2][0] = 'X'
>>> t[2][1] = 'Y'
>>> t
('a', 'b', ['X', 'Y'])

    2.关于可变&不可变
    str是不变变量，但是list是可变变量。例如，a是一个list变量，a.sort将a本身进行了排序并且覆盖了原先的a, 但是若a是一个str, a.replace('x','y')只是生成一个新字符串并返回，str变量a作为指针指向的变量依旧是未替代前的那一个。

    3.关于dit和set
    key都要求唯一&不可变 dit 用花括号，而且有冒号，无需表明dit set 用圆括号

    >>> dit_test= {'Onel': 1, 'Two': 2}
>>> set_test = set([1,2,3])
>>> dit_test 
{'Onel': 1, 'Two': 2}
>>> set_test
{1, 2, 3}

    使用key-value存储结构的dict在Python中非常有用，选择不可变对象作为key很重要（不管是指向不可变，还包括指向的内容也不可变），最常用的key是字符串。
    Actually, set()只是一个函数，函数的参数是一个list，函数的返回值是set类型的对象，与tuple类似，再往后看，参数只要求是Iterable就可以 ----by 廖雪峰

    关于条件循环

    for x in ...循环就是把每个元素带入变量x，然后执行缩进块的语句。

    while循环是指只要条件满足就不断循环

    Slice 取list或tuple或字符串的部分元素时，用循环虽然不是最笨但也很繁琐，所以Slice就出现了 - 正数的第一个索引是0（前11-20 ） - 倒数的第一个索引是-1(最后10个) - 前10个数，间隔取（每隔两个数取一个）

    Iteration - Python高级之处（in）e.g. dict上的迭代 - 判断对象可迭代 - 生成下标循环，索引元素对

    List Comprehensions - 生成具有函数基因的列表[x * x for x in range(1,11) if x % 2 == 0] - 两个循环的全排列 - 一个循环可以使用多个变量

    Generator 如果已经知道了一个非常大的列表暗藏的规律，就不必创建完整的列表浪费存储空间而是可以一边循环一边生成。 - Generator和函数在执行和返回时的区别： - 函数：顺序执行&return返回|最后一行返回 - generator：next()执行，yield返回，再回到上次yeild处 - Generator一般用for来输出结果 - 用for循环时，通过捕捉StopIeteration来获得返回值 - e.g. Fibonacci

    Ieterator - 被用于for循环 - 被next()调用 - Iterable->Iterator: for循环的本质可以看做调用next()

    5月9日 函数式编程

    默认参数

    为什么要设计str、None这样的不变对象呢？

    因为不变对象一旦创建，对象内部的数据就不能修改，这样就减少了由于修改数据导致的错误。此外，由于对象不变，多任务环境下同时读取对象不需要加锁，同时读一点问题都没有。

    可变参数（参数的数量可以改变） - 定义可变参数和定义list或tuple型的参数之间的区别 - 把list元素和tuple元素变成可变参数传入函数中，因为函数调用时设计可变参数（list元素中的某个值代表的是一个可变参数的值）

    关键字参数（用作扩展函数，e.g. 除姓名和年龄必填其余都是可选）

    可变参数允许你传入0个或任意个参数，这些可变参数在函数调用时自动组装为一个tuple。而关键字参数允许你传入0个或任意个含参数名的参数，这些关键字参数在函数内部自动组装为一个dict。 by 廖学峰（ps：总结得真好：）

    • 命名关键字参数 分隔符的要与不要 def person(name, age, *, city='Beijing', job):

    在Python中定义函数，可以用必选参数、默认参数、可变参数、关键字参数和命名关键字参数，这5种参数都可以组合使用。

    但是请注意，参数定义的顺序必须是：必选参数、默认参数、可变参数、命名关键字参数和关键字参数。 def f1(a, b, c=0, *args, **kw): def f2(a, b, c=0, *, d, **kw): f1(*args, **kw)与f2(*args, **kw)都是可以的--（竟然有个通用的函数调用模式）
