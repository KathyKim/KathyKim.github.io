《Doing Data Science》

中文版为《数据科学实战》

1.简介：

  • 官方

How can you learn this wide-ranging, interdisciplinary field? With this book, you’ll get material from Columbia University’s "Introduction to Data Science" class in an easy-to-follow format.

Each chapter-long lecture features a guest data scientist from a prominent company such as Google, Microsoft, or eBay teaching new algorithms, methods, or models by sharing case studies and actual code they use. You’ll learn what’s involved in the lives of data scientists and be able to use the techniques they present.

  • 岛主夫人随便说两句

这本书是哥大所开课程"Introduction to Data Science"的讲义集，虽然哥大统计专业的中国人多到能让老美觉得他在中国留学，而且老美不怎么认可这个专业的学生，但是他们大都是回国干一番事业的所以也没有关系。

最重要的是，一分学费一分货呀，例如这门课所请的嘉宾很厉害。

非常遗憾这两本书在豆瓣上一直处于极少人看的状态，我也只能用「是金子总会发光的」来安慰这本极其优秀的「数据科学」主题的书。

2.深刻的几点洞见

  • Where and what domain data science happens in is not the issue— rather, defining it as a “problem space” with a corresponding “solution space” in algorithms and code and data is the key.( 关键是通过算法和代码定义一个从「问题空间」到「解法空间」的映射，而数据，则是「key」）

  • The problem, as I see it, is not one of replacing human ingenuity by machine intelligence, but one of assisting human ingenuity by all conceivable tools of computer science and artificial intelligence , in particular aiding with the improvisation of search tools and with keeping track of the progress of an analysis. ——by Matthew Jones (机器智能的辅助性，更明确地说，是能够让人「即兴」地实践自己的想法并且获得反馈结果）

  • No data scientists need ever confuse the possibility of using a technology with the necessity of using it. In the sparse space of the infinite things one might do with data , only a few dense(and interesting) ethical pockets deserve out energy. ——by Matthew Jones （要克服自己的虚荣心呢，千万千万不要将使用一项『高贵』技术的「可能性」与「必要性」混为一谈）
